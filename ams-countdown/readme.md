# AMS Countdown

**Creat de Raluca Manea – [amoos.ro](https://amoos.ro)**

---

## 📋 Descriere

AMS Countdown este un plugin simplu pentru WordPress care adaugă un shortcode `[ams_countdown]` într-o pagină sau un articol.  
Acesta creează automat un cronometru regresiv (countdown) către data și ora salvate cu ajutorul pluginului **AMS - Replace Date and Hour Range**.

---

## 🚀 Caracteristici

- Afișează un countdown elegant și responsiv pe website.
- Se conectează automat la valorile din **AMS - Replace Date and Hour Range**.
- Nu necesită configurare suplimentară.
- Afișare timp rămas în zile, ore, minute și secunde.
- Stil simplu și curat, ușor de personalizat prin CSS.

---

## 🛠 Instalare

1. Asigură-te că pluginul **AMS - Replace Date and Hour Range** este instalat și activ.
2. Încarcă folderul `ams-countdown` în directorul `/wp-content/plugins/`.
3. Activează pluginul din panoul de administrare WordPress.
4. Inserează shortcode-ul `[ams_countdown]` oriunde dorești să apară cronometru.

---
